## 🔧 如需调整

### 如果需要严格单屏适配（1080p 无滚动）

修改 app.py，将高度改回：

```python
# 方案 1: 保持宽度优化，减小高度
with st.container(height=600):  # 从 700 改为 600
    pdf_viewer(input=pdf_bytes, width=1200, height=600)

# 方案 2: 恢复原始尺寸
with st.container(height=520):  # 恢复原始
    pdf_viewer(input=pdf_bytes, width=1200, height=520)
```
